Efeito Parallax

O efeito parallax é uma técnica de web design que faz com que uma imagem de fundo se movimente de forma mais lenta que outras imagens à frente. Isso gera um efeito 3D conforme o usuario navega pela página, criando desse jeito uma sensação de profundidade.

O efeito paralax é baseado em uma ilusão de ótica, pois o olho humano supõe que os objetos que estão perto são maiores em comparação com os que estão afastados.

Essa aplicação web, tem como objetivo o aprendizado de fundamentos de HTML5, CSS3, JavaScript e o Efeito Parallax. Para o desenvolvimento dessa aplicação, foi utilizado como base o vídeo: Parallax Scrolling Website | How to Make Website using Html CSS & Javascript do canal: Online Tutorials.
